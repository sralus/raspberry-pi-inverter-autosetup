# PI30 / WR1

Dieser Ordner enthält die Projektbasis für PI30 / WR1.

## Inhalt

- runtime/ -> Reader, Builder, Controller
- ui/ -> Built-in UI Dateien
- templates/ -> systemd-Vorlagen für Reader, Builder, Timer und UI

## Ziel

- Installation über python3 setup_pi30.py
- built-in UI als Standard
- kein externer Webserver als Pflicht
