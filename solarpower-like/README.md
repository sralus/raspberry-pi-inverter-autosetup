# PI18 / WR2

Dieser Ordner enthält die Projektbasis für PI18 / WR2.

## Inhalt

- runtime/ -> Reader, Builder, Controller, UI-API
- ui/ -> Built-in UI Dateien
- templates/ -> systemd-Vorlagen für Reader, Builder, Timer und UI

## Ziel

- Installation über python3 setup_pi18.py
- built-in UI als Standard
- kein externer Webserver als Pflicht
